- 値渡し：数字などのコピーの時に、値自体をコピーすること
- 参照渡し：リストのコピーなどの時に、その id を渡すこと

Rust では、コピーした変数じゃ所有権を失い、その値へのアクセス権限を失う。Python の場合では、2つの変数から値にアクセスできるようになているので、***バグる***。=>**これが Rust では起こらない！！便利！！暗示するより明示する方がよい**
